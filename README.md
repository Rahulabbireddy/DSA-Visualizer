# 📚 DSA Visualizer

## Overview
DSA Visualizer is an interactive web application that demonstrates Data Structures and Algorithms using animations.

## Features

### Data Structures
- Array
- Stack
- Queue
- Linked List
- Binary Search Tree
- AVL Tree
- Heap
- Graph

### Algorithms
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort
- Linear Search
- Binary Search
- Jump Search
- Interpolation Search

## Technologies
- HTML5
- CSS3
- JavaScript

## How to Run

1. Open `loader.html`
2. Wait for loading
3. Explore all visualizers
